# EDM Festival Website

Frontend project for an EDM music festival website, built as a practice project to improve modern frontend workflows and tools.

##  Technologies
- HTML5
- Sass (SCSS)
  - Variables
  - Mixins
  - Modular structure with `@use`
- JavaScript (Vanilla JS)
- npm
- Gulp

##  Features
- Responsive design (mobile, tablet, desktop)
- Navigation menu with active section highlight on scroll
- Image gallery generated dynamically with JavaScript
- Modal image viewer with fade-in / fade-out animation
- Organized and scalable Sass architecture
